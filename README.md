# Albion-Radar (Holo)

Albion Online Player and Harvestable Radar

# Getting Started

-   Download project as source code with git

```
git clone https://github.com/Macsimka/Albion-radar.git

```

-   Launch Holo.sln
-   Compile and Have Fun
